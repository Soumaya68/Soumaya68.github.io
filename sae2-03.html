Application web de supervision aéroportuaire

La SAE 2.03 nous a demandé de concevoir une application web complète de gestion du trafic aérien entre aéroports. Notre système permet aux gestionnaires de superviser vols, pistes, avions et compagnies, avec des vérifications métier intelligentes.

L'architecture repose sur deux VM Linux séparées — serveur web (Gunicorn + Nginx) et base de données MariaDB distante — proche d'un environnement professionnel réel.

// ENGLISH SUMMARY
This project involved building a complete web application for managing airport operations using Django 5.2 and MariaDB, deployed on two Linux VMs (Nginx + Gunicorn). As Frontend Developer & UX-UI Expert, I designed all interfaces, CRUD forms and Django templates. The system manages airports, runways, aircraft and flights with intelligent business logic including compatibility checks and temporal constraints.
