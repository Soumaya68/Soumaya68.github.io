Interface de gestion du trafic aérien

Développement d'une solution web complète pour la gestion des opérations aéroportuaires. Cette application permet aux gestionnaires d'aéroports de superviser efficacement le trafic aérien, les pistes et les vols avec des vérifications automatiques de compatibilité.

Architecture Django professionnelle

Backend robuste développé avec Django 5.2.3, utilisant l'architecture MVT et l'ORM Django pour une gestion optimisée des données relationnelles.

Django 5.2.3 Python ORM

Gestion relationnelle complexe

Base de données MariaDB avec 6 entités principales : Aéroport, Piste, Compagnie, TypeAvion, Avion et Vol, avec contraintes d'intégrité et vérifications automatiques.

MariaDB Relations 1:N Contraintes

Fonctionnalités métier avancées

CRUD complet avec vérifications intelligentes : compatibilité piste/avion selon la longueur requise, disponibilité des créneaux, import/export de vols en masse via CSV.

CRUD Import CSV Validation

Chronologie du développement

Phase 1 - Base de données

Création de la structure MariaDB avec toutes les tables et insertion de données de test arbitraires.

Phase 2 - Intégration Django

Génération automatique du models.py avec python manage.py inspectdb.

Phase 3 - Interface de consultation

Développement des pages d'affichage pour visualiser les données (listes d'aéroports, avions, vols).

Phase 4 - CRUD complet

Ajout des fonctionnalités de création, modification et suppression pour toutes les entités.

Phase 5 - Fonctionnalités avancées

Import CSV, vérifications de compatibilité piste/avion et logique métier spécialisée.

Phase 6 - Déploiement

Architecture de production avec VM Linux, nginx et Gunicorn.

Mon ressenti et apprentissages humains

Adaptation au travail collaboratif imposé

Ce projet a été particulièrement challengeant sur le plan humain. Habituée à prendre naturellement le leadership dans les projets de groupe, j'ai dû apprendre à m'adapter à une dynamique différente avec une équipe qui m'était imposée.

Au début, le rythme était très intense et j'avais du mal à suivre certaines étapes techniques que mes camarades maîtrisaient déjà. Cette situation était frustrante car je ne comprenais pas forcément les choix d'architecture système que Kenzy et Yoann implémentaient.

Découverte d'un nouveau style de leadership

Travailler avec Kenzy, qui prenait naturellement les devants sur les aspects techniques complexes, a été très enrichissant. Au lieu de me sentir mise à l'écart, j'ai découvert qu'il pouvait être rassurant de laisser quelqu'un d'autre guider certaines parties du projet.

Cette expérience m'a appris qu'il existe différentes façons de contribuer efficacement à un projet, et que le leadership peut être partagé selon les domaines d'expertise de chacun.

Communication et bienveillance d'équipe

Le tournant du projet s'est produit quand j'ai pu exprimer mes difficultés à mes coéquipiers. Leur écoute et leur patience ont été remarquables. Ils ont pris le temps de m'expliquer les étapes techniques complexes sur lesquelles je n'avais pas encore mis les mains.

Cette bienveillance m'a permis de rattraper le retard et de contribuer pleinement au projet. Je les remercie pour cette attitude qui a transformé un moment difficile en véritable apprentissage collaboratif.

Enrichissement technique et humain

Au final, ce projet m'a fait découvrir énormément de possibilités techniques que je n'avais pas envisagées : l'architecture Django avancée, le déploiement avec nginx et Gunicorn, la gestion de bases de données distantes avec MariaDB.

Plus important encore, j'ai appris que la diversité des approches et des personnalités dans une équipe est une force quand elle est bien gérée. Cette expérience a enrichi ma vision du travail collaboratif et m'a donné confiance pour affronter de futurs défis techniques.

English Summary

Airport Traffic Management System

Complete web solution for airport operations management

Project Overview

This project involved developing a comprehensive web application for managing airport traffic operations. Built as part of SAE 2.03 (Implementing IT Solutions), the system enables airport managers to efficiently supervise air traffic, runways, and flights with automated compatibility checks and intelligent scheduling.

Technical Stack

  • Backend: Django 5.2.3 with MVT architecture
  • Database: MariaDB with complex relationships
  • Frontend: HTML5, CSS3, responsive design
  • Deployment: Linux VMs, nginx, Gunicorn

Key Features

  • Complete CRUD operations for all entities
  • Aircraft-runway compatibility verification
  • CSV import/export for bulk flight management
  • Real-time scheduling conflict detection
