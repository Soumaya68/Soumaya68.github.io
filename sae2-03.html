🇬🇧 English Summary
Air Traffic Management Web Application

This project involved building a complete web application for managing airport operations using Django 5.2 and MariaDB, deployed on two Linux VMs (Nginx + Gunicorn). As Frontend Developer & UX-UI Expert, I designed all interfaces, CRUD forms and Django templates. The system manages airports, runways, aircraft and flights with intelligent business logic including compatibility checks and temporal constraints.
