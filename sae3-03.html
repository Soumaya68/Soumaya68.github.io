Vue d'ensemble

Infrastructure opérateur réelle

Ce projet reproduit l'architecture réseau d'un véritable FAI : deux entreprises (UC Exchange et ABC Conseil) réparties sur plusieurs villes, interconnectées via un réseau opérateur simulé sous GNS3. Chaque site dispose d'une architecture LAN hiérarchique reliée au cœur réseau.

Le travail est réparti en trois parties complémentaires. J'ai pris en charge la partie FAI — concevoir le cœur réseau opérateur avec les systèmes autonomes, le routage dynamique, MPLS, BGP et les VRF pour garantir isolation et tolérance aux pannes.

Retour d'expérience

Ce que j'en retiens

Ce projet a été l'un des plus techniques que j'ai réalisés. La partie FAI m'a demandé une grande rigueur, notamment dans la configuration de BGP et la compréhension de MPLS. J'ai dû reprendre plusieurs configurations, ce qui m'a permis de développer une méthodologie efficace pour diagnostiquer et résoudre les problèmes réseau. Ce projet m'a apporté une vision concrète des architectures opérateurs.

// ENGLISH SUMMARY
This project focused on designing a secure multi-site network infrastructure for two companies. I was responsible for the ISP (FAI) part, which involved building a core network using MPLS, BGP, and VRF technologies to ensure secure and isolated communication between sites. I configured routing protocols (RIP, OSPF), deployed MPLS, implemented MP-BGP, and validated redundancy mechanisms. This project strengthened my skills in advanced networking and gave me practical insight into real-world ISP infrastructures.
