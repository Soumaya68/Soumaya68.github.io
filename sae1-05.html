Vue d'ensemble

Un outil de reporting complet

Ce projet consistait à concevoir un outil capable d'analyser toute l'arborescence d'un disque dur, de localiser les fichiers dépassant un seuil de taille donné, et de présenter les résultats dans une interface graphique interactive. J'ai réalisé l'intégralité du code de manière autonome.

L'outil est orchestré par un script PowerShell qui enchaîne trois scripts Python : sélection du répertoire, analyse et export JSON, puis affichage graphique.

// ENGLISH SUMMARY
This project involved building a complete Python reporting tool to locate large files on a disk. Using pathlib for recursive directory traversal, files are sorted by size, filtered by threshold, and exported to JSON. A PyQt5 graphical interface displays the results as a pie chart with colored legends, checkboxes for file selection, and a button to auto-generate a PowerShell deletion script. The entire pipeline is orchestrated by a PowerShell script.
