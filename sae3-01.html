Vue d'ensemble

Un système IoT de bout en bout

Ce projet répond à un cahier des charges réaliste : concevoir une prise électrique connectée en Wi-Fi capable de contrôler des équipements à distance, avec programmation horaire, mesure de température et notifications en temps réel.

L'architecture combine électronique embarquée, infrastructure réseau et développement logiciel — couvrant l'ensemble du périmètre d'un système IoT professionnel, de la conception matérielle jusqu'à l'exploitation des données.

// ENGLISH SUMMARY
This project involved designing and implementing a smart Wi-Fi connected power outlet. We developed a complete IoT system including a hardware device (ESP8266), a web server (Raspberry Pi), a web interface, and an Android application. The system allows remote control, scheduled operations, temperature monitoring, and push notifications. This project helped me understand how connected systems work, from hardware design to network communication and application development.
