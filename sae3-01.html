Contexte

Concevoir et déployer une prise connectée intelligente

Ce projet avait pour objectif de concevoir un système IoT complet permettant de contrôler à distance des équipements électriques via une interface web et une application mobile. L'architecture combine électronique, réseaux et développement logiciel.

Système de transmission

Communication sans fil Wi-Fi entre équipements

Application communicante

Interfaces web et mobile pour piloter le système

Sécurité & supervision

Mécanismes de sécurisation et de notification

Architecture

Système IoT distribué

L'architecture repose sur trois couches interconnectées : les objets connectés, le serveur central et les interfaces utilisateur.

🔌 Prise Wi-Fi (ESP8266)

Carte électronique avec module Wi-Fi intégré pour le contrôle local

🖥️ Serveur Raspberry Pi

Hébergement du serveur web et du broker MQTT central

🌐 Interface web

Contrôle à distance, programmation horaire, supervision

📱 Application Android

Reprend les fonctionnalités principales sur mobile

Fonctionnalités avancées : programmation horaire, mesure de température via capteur, notifications (changement d'état, seuil dépassé) et sécurisation des accès.

Ce que j'en retiens

Ce projet m'a permis de découvrir le fonctionnement d'un système IoT complet, en combinant électronique, réseau et développement. J'ai particulièrement apprécié la diversité des compétences mobilisées et la complexité de la communication entre composants. Il a renforcé ma capacité à analyser un cahier des charges et à proposer des solutions techniques adaptées.

🇬🇧 English Summary
Smart Wi-Fi Connected Power Outlet — IoT System

This project involved designing and implementing a complete IoT system including a hardware device (ESP8266), a web server (Raspberry Pi), a web interface, and an Android application. The system allows users to control devices remotely, schedule operations, monitor temperature, and receive notifications. This project helped me understand how connected systems work, from hardware design to network communication and application development.
